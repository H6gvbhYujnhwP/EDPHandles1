# Handles Page Structure

## Page Purpose

The Handles page will become the first true product-family page within the EDP Handles site. Its purpose is to translate Cosma’s manufacturer-led catalogue into a curated, UK-facing distributor experience that remains consistent with the existing **Neo-Atelier Italian Editorial** design system.

## Design Direction

The page should retain the current site language: dark showroom backgrounds, brass-accent highlights, serif-led headlines, clean Manrope support typography, calm motion, and asymmetrical editorial spacing. Nano Banana imagery should be reserved for the brand-led opener or transitional editorial panels, while real product images should carry the product sections.

## Recommended Page Sections

| Section | Purpose | Content type |
|---|---|---|
| Hero | Introduce the handles offering in a premium, distributor-led way | Nano Banana hero image, brand copy, CTA into categories |
| Category Overview | Present the main curated handle families for UK browsing | Category cards with real product imagery |
| Featured Ranges | Showcase selected lead handles or grouped ranges | Range cards or product tiles |
| Materials & Finishes Teaser | Connect handles to finish options without building the full finishes page yet | Short editorial strip with finish cues |
| Trade Specification Panel | Reinforce UK distributor and project support positioning | Copy + CTA |
| Related Navigation | Lead users onward to Finishes, Open Systems, Trade, and Contact | Internal links |

## Proposed Handle Categories for Phase One

The first version should use clear UK-facing category labels mapped from Cosma’s structure rather than exposing Italian internal taxonomy directly.

| EDP category label | Based on Cosma categories |
|---|---|
| Bridge Handles | Ponte, Ponte Modulari |
| Integrated Handles | Presa Chiusa, Presa Chiusa Modulare, Integrate, Integrate Modulari |
| Vertical Handles | Verticali, Verticali Integrate |
| Appliance Handles | Elettrodomestici |
| Knobs & Small Hardware | Pomoli |
| Special Designs | Speciali |

## Content Fields Per Category or Range

Each showcased item should eventually support the same internal content structure so future pages can scale cleanly.

| Field | Description |
|---|---|
| Name | Category or range title shown on page |
| Source category | Original Cosma classification |
| Short description | UK-facing edited copy for design/trade audience |
| Image set | Lead image plus optional supporting gallery images |
| Material cues | Optional material references if known |
| Finish cues | Optional finish references if known |
| CTA | Enquiry or trade CTA |

## Implementation Principle

This page should be treated as the repeatable template for future product-family pages. Once this structure is approved and implemented, the same layout logic can be adapted for Open Systems, Finishes, and other sections with minimal design drift.
